# NAI-Helper
NAI Helper Android release
NAI Helper는 NOVEL AI 프롬프트 제작/관리 및 이미지 생성을 위한 통합 관리 어플리케이션입니다.

- 본 앱은 NovelAI 공식 클라이언트가 아닙니다.
- NovelAI 계정 토큰은 사용자가 직접 입력해야 합니다.
- 입력한 토큰과 앱 데이터는 사용자 기기 내부에 저장되며, 개발자는 이를 알 수도, 활용할 수도 없습니다.
- 반복 이미지 생성 기능 사용 시 과도한 요청을 피해주세요.
- 사용 환경에 따라 오류가 발생할 수 있습니다.

## 설치 방법

1. 아래 APK 파일을 다운로드합니다.
2. Android 기기에서 APK를 실행합니다.
3. 알 수 없는 앱 설치 허용 안내가 나오면 허용합니다.
4. 최초 실행 시 사용자 약관에 동의한 뒤 사용합니다.

#이용약관
# Nai Helper 사용자 약관 및 이용 안내

최종 수정일 : 2026-05-09

## 제1조 (목적)

본 약관은 Nai Helper(이하 "앱")의 사용 조건 및 책임 범위를 안내하기 위한 목적으로 작성되었습니다.

본 앱은 AI 이미지 생성과 관련된 프롬프트 관리, 태그 관리, 메타데이터 분석 및 작업 편의 기능을 제공하는 개인 제작 도구입니다.

---

## 제2조 (앱의 성격)

Nai Helper는 다음 기능을 제공합니다.

* 프롬프트 작성 및 관리
* 태그 사전 및 와일드카드 관리
* 이미지 메타데이터 기반 프롬프트 추출
* AI 이미지 생성 보조 기능
* 사용자 로컬 환경 기반 데이터 관리

본 앱은 AI 생성 모델 자체를 제공하지 않으며, 이미지 생성 서비스 서버를 운영하지 않습니다.

---

## 제3조 (NovelAI 연동)

앱의 일부 기능은 사용자가 직접 입력한 NovelAI 계정 토큰을 이용하여 동작합니다.

* 사용자는 본인의 계정 및 API 토큰만 사용해야 합니다.
* 타인의 계정 공유, 무단 사용, 판매 행위는 금지됩니다.
* 사용자는 NovelAI의 이용약관 및 정책을 준수해야 합니다.
* 본 앱은 NovelAI의 공식 클라이언트가 아닙니다.

앱 제작자는 NovelAI 서비스 정책 변경, 계정 제한, 서비스 이용 제한 등에 대해 책임지지 않습니다.

---

## 제4조 (자동 생성 기능)

앱에는 반복 이미지 생성 기능(예: 무한 생성 기능)이 포함될 수 있습니다.

사용자는 다음 사항을 준수해야 합니다.

* 과도한 자동 요청 금지
* 서비스 서버에 과도한 부하를 유발하는 사용 금지
* 자동화 기능 악용 금지

앱은 요청 제한(429 Too Many Requests) 발생 시 자동 대기 또는 기능 제한이 적용될 수 있습니다.

---

## 제5조 (태그 데이터 및 데이터베이스)

앱에 포함된 일부 태그 데이터베이스 및 분류 데이터는 외부 공개 자료 및 제공자의 허가를 받아 사용될 수 있습니다.

데이터 제공자 정보는 앱 내 정보 화면에 표시됩니다.

사용자는 앱 내 데이터를 무단 재배포하거나 상업적으로 이용해서는 안 됩니다.

---

## 제6조 (콘텐츠 및 책임)

앱은 프롬프트 및 태그 관리 도구이며, 사용자가 입력하거나 생성한 콘텐츠에 대한 책임은 사용자 본인에게 있습니다.

사용자는 다음과 같은 용도로 앱을 사용해서는 안 됩니다.

* 불법 콘텐츠 제작
* 타인의 권리를 침해하는 행위
* 관련 법령을 위반하는 행위

앱 제작자는 사용자가 생성한 콘텐츠에 대해 책임지지 않습니다.

---

## 제7조 (데이터 저장)

앱 데이터는 기본적으로 사용자 기기 내부에 저장됩니다.

앱 제작자는 사용자의 프롬프트, 생성 이미지, 계정 토큰 등의 데이터를 외부 서버에 수집하거나 저장하지 않습니다.

단, 사용자의 기기 환경 또는 외부 서비스 정책에 따라 데이터 손실이 발생할 수 있으며, 이에 대한 책임은 사용자 본인에게 있습니다.

---

## 제8조 (면책사항)

앱 제작자는 다음 사항에 대해 책임지지 않습니다.

* 외부 서비스 정책 변경
* NovelAI 서비스 제한 또는 계정 문제
* 데이터 손실
* 사용자 기기 문제
* 사용자 생성 콘텐츠로 인해 발생하는 문제

---

## 제9조 (약관 변경)

본 약관은 필요에 따라 변경될 수 있으며, 변경 시 앱 내 또는 배포 페이지를 통해 안내될 수 있습니다.
